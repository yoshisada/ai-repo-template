## Branch naming convention

Use `feat/`, `fix/`, `build/`, or `docs/` prefixes for branches. The hook blocks pushes to branches without one of these prefixes, because skipping the convention silently breaks our CI gate that maps branch prefix → workflow.

## Commit message style

Use imperative mood. Subject under 72 chars. Body wraps at 80.
