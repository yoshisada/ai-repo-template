## The closed feedback loop is the product

We capture friction, distill it into PRDs, ship the PRDs as features, and
retrospect on what worked. Each iteration improves the next.

## Context-informed autonomy

The system deliberates autonomy-vs-escalation using prior decisions as
precedent — not blanket "flag every judgement call".

## Plugins ship independently

Wheel is plugin-agnostic infrastructure. Each plugin's release cadence
is its own.
