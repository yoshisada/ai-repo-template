# File Census

This repository contains 312 Markdown files, 37 shell scripts, and 23 JSON files (excluding node_modules and .wheel directories). The high proportion of Markdown files reflects the plugin's skill-and-agent architecture, where each skill, agent, and template is defined as a Markdown document. The shell scripts serve as hooks, build utilities, and workflow automation, while the JSON files cover package manifests, plugin configurations, and workflow definitions.
