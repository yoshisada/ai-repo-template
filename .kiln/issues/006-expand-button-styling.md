## Description

The expand button (+) in the right sidebar step list for workflow expansion has poor visual styling. It should be more prominent and visually distinct from other UI elements.

## Impact

Users may not notice or understand the expand button functionality due to its current styling.

## Suggested Fix

Improve the expand button CSS to be more visible - consider using a distinct color, better sizing, hover effects, and possibly a different icon or label format.
